# Zyra-Fashion-AI

Voice-activated AI fashion assistant using YOLOv8, Google Gemini, 
and fal.ai FLUX for real-time outfit analysis, skin-tone-based 
recommendations, and AI outfit visualization. Supports 30+ occasions 
and Indian/Western dress types.

## ✨ Features
- 🎤 Voice activation with wake word "Zyra"
- 📷 Real-time body detection using YOLOv8
- 🤖 Outfit analysis using Google Gemini AI
- 👗 Skin-tone-based outfit recommendations
- ⭐ Outfit rating with improvement tips
- 🎨 AI outfit visualization using fal.ai FLUX
- 🇮🇳 Supports Indian and Western dress types
- 🎉 30+ occasions including Diwali, Onam, Wedding and more

## 🛠️ Tech Stack
- Python, Flask, YOLOv8, Google Gemini API
- OpenRouter API, fal.ai FLUX, OpenCV
- JavaScript, HTML/CSS, Web Speech API
- Google Colab, ngrok

## 🔑 API Keys Required
- Google Gemini API Key
- OpenRouter API Key
- fal.ai API Key
- ngrok Auth Token

## 🚀 Run
- Open in Google Colab
- Add API keys in Colab Secrets 🔑
- Run all cells sequentially
- Enter ngrok token when prompted
- Open the public URL in browser
